

Задание: Помогите слуге правильно составить иерархию и подготовить отчет для короля
В pollResults входные данные.
В классе UnluckyVassal выполняется обработка входных данных.
Определяется отношение руководитель-подчиненный для каждого из списка.
В классе SortedServant выполняется сортировка данных.
Организован вывод данных в двух форматах
1. в отсортированном виде
2. в иерархическом виде

Класс OutputHierarchy для вывода данных в иерархическом виде.
 - первый уровень Король
 - второй уровень все, кто подчиняется Королю
 - третий уровень все, кто подчиняется членам предыдущего уровня
  - и т.д.
